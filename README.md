# 🐾 AI Pet

An interactive AI-powered virtual pet that lives in your web browser. Feed it, play with it, and watch it grow!

## Tech Stack

- **React** + **TypeScript** — UI framework
- **Vite** — Build tooling & dev server

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
